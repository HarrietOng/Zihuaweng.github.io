RCNN RCNN 是RCNN网络的第一。分为三个部分：生成各种类别的候选框， 使用CNN对同等大小的图片进行特征提取， 每一种种类的SVM 训练： 1. 输入图片 2. 在图像中确定约2000个候选框 (使用选择性搜索) 　　2. 每个候选框内图像块缩放至相同大小，并输入到CNN内进行特征提取 　　3. 对候选框中提取出的特征，使用分类器判别是否属于一个特定类 　　4. 对于属于某一特征的选框，用回归器进一步调整其位置 测试： 1. 输入图片 2. SS 选取2000个候 3. 每个候选框转换成一样小， 经过CNN提取特征 4. 每个候选框经过每个类别训练好的SVM 5. Fast RCNN 　　1. 在图像中确定约1000-2000个候选框 (使用选择性搜索) 　　2. 对整张图片输进CNN，得到fature map 　　3. 找到每个候选框在feature map上的映射patch，将此patch作为每个候选框的卷积特征输入到SPP layer和之后的层 　　4. 对候选框中提取出的特征，使用分类器判别是否于一个特定类 　　5. 对于属于某一特征的候选框，用回归器进一调整其位置 Faster RNN 　　1. 对整张图片输进CNN，得到featuremap 　　2. 卷积特征输入到RPN，得到候选框的特征信息 　　3. 对候选框中取出的特征，使用分类器判别是否属于一个特定类 　　4. 对于属于某一特征的候选框，用回归器进一步调整其位置
